# README #

Translate selected text easy

## Use ##

```
make dir easytrans
$ mkdir /path/easytrans
$ cd /path/easytrans

clone Appmenu
$ git clone https://bitbucket.org/remasik/easytrans.git

rights
# chmod +x easytrans.py
$ ./easytrans.py -t
Use: select text and run easytrans
```
> You can assign a hotkey for start an easytrans
